Spatial System

8px grid · 22 tokens from 2px to 256px

All spacing is based on an 8px base unit. Token naming follows the Tailwind convention: --space-1 = 4px, --space-2 = 8px, --space-4 = 16px. Half-steps (--space-0-5, --space-1-5) exist for tight UI contexts. This system ensures visual rhythm across all three panels of the three-panel layout.

Motion Principles

Token streaming · Panel transitions · Editor micro-interactions
✦ Token Streaming Animation
Each LLM token fades in with a subtle blur-to-sharp transition over 80ms. Tokens appear sequentially with staggered delay. The cursor blinks at 1s intervals between bursts.

⟷ Panel Slide Transition
When panels collapse at tablet/mobile breakpoints, the active panel slides in from the right using translateX with cubic-bezier decelerate easing over 320ms. No opacity transition — pure transform only for GPU compositing.

Rule Requirement Implementation Test
Focus Rings All interactive elements must have visible focus ring 3px offset, crystal-400 color, 2px border-radius. Never use outline:none without alternative. Tab through full app flow
Color Independence Never use color alone to convey state Confidence badges always include text label. Processing stages always include icon + label. Grayscale screenshot test
Motion Reduced Respect prefers-reduced-motion All animations gated behind @media (prefers-reduced-motion: no-preference). Token streaming degrades to instant render. Enable OS reduced motion
Target Sizes Minimum 44×44px touch targets on mobile All buttons and icons pad to minimum 44px on mobile breakpoint. Icon buttons use btn-icon with padding expansion. Mobile touch test
ARIA Labels All icon-only buttons need aria-label aria-label on all toolbar-btn. Processing stages use aria-live="polite" for status updates. Chat uses aria-live="assertive". Screen reader (NVDA/VoiceOver)
Headings Hierarchy Logical heading structure without skipping levels H1 = page title, H2 = section, H3 = subsection, H4 = card title. Decorative headings use aria-hidden. Heading map browser extension
Text Resize Content readable at 200% zoom without horizontal scroll All layouts use relative units (rem, %). Panel layout reflows to stacked at 200% zoom equivalent. Browser zoom to 200%
